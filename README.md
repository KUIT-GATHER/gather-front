# Gather Frontend

Gather 서비스의 프론트엔드 프로젝트입니다.

## Tech Stack

- React
- TypeScript
- Vite
- React Router
- TanStack Query
- Tailwind CSS
- Zustand
- React Hook Form
- Zod

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

```bash
npm run dev      # 개발 서버 실행
npm run build    # 타입 검사 및 프로덕션 빌드
npm run lint     # ESLint 검사
npm run preview  # 빌드 결과 미리보기
```
